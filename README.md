qaAccordion
===========

This is a simple jQuery library to make the accordion effect.

The basic html is as fallows:

<code>
<div id="qaAccordion_wrapper">
	<div class="qa_item">
		<div class="qa_question qa_off">
			qa Title
		</div>
		<div class="qa_answer">
			Your anwser goes here...
		</div>
	</div>
</div><!-- qaAccordion_wrapper -->
</code>
