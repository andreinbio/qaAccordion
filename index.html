Welcome to qaAccordion site.

This is a simple jQuery library that creates the accordion effect on selected elements.

The basic html that the library expects is:

`<div id="qaAccordion_wrapper">  
    <div class="qa_item">  
        <div class="qa_question qa_off">  
            <!-- here is the title/question ... -->  
            qa Title  
        </div>  
        <div class="qa_answer">  
            <!-- here is some text -->  
            Some text/answer ...  
        </div>  
    </div>  
</div><!-- qaAccordion_wrapper -->`
